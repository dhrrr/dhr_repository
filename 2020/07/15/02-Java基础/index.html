发表于 2020-07-15 | 分类于 Java入门
字数统计: 385 | 阅读时长 ≈ 1

变量

  • 操作变量
  • 变量用于存储内容，在计算机中等同一个有命名的存储地址
  • 变量：是解决内容复用的；有名称
  • Java中变量的内容，就是对象
  • eg：String name; //声明了一个名为name，类型为String的变量
  • 变量首字母一般小写

关键字

  • 不允许用来声明的名称：
    • public / class / void / int 等等
    • eg：String public; //❌

赋值

  • 声明变量后，往变量里存储数据的行为叫赋值、
  • 重复赋值：变量赋值后，可再次被修改

数据类型

  • 字符串 String
    • 字符串+数字：会变成字符串
  • 数字
    • 小整数 int
      • 最小值：Integer.MIN_VALUE = -2的31次方
      • 最大值：Integer.MAX_VALUE = 2的31次方-1
    • 大整数 long
      • 最小值：Long.MIN_VALUE = -2的63次方
      • 最大值：Long.MAX_VALUE = 2的63次方-1
      • 注意：long类型赋值时，数字后要跟一个小写的l（L）。eg：long num = 78900000123l;
    • 浮点数 double
  • 布尔 boolean
    • 布尔值只能是true或false

运算符

  • 加法 +
  • 减法 -
  • 乘法 *
  • 除法 /
  • 整除 %
int now = 59;
System.out.println(now/60);

结果为：0    //Java执行整数除法，当两操作数都是整数时，结果自动变整数（直接去掉了小数点值）
int now = 59;
System.out.println(100*now/60);

结果为：98    //计算百分比
double now = 59;
double result = now/60;
System.out.println(result);

结果为：0.983333333333333    //浮点类型
