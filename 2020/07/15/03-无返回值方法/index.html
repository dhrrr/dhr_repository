数学函数（生成随机数）

double value = Math.random();    //0.40068331197074114   0.21177156791789276

  • Math.random()方法用于返回一个随机数

  • 随机数范围为：0.0=<X<1.0

    double value = Math.random();
int newValue = (int)(value * 10);    //把随机的结果*10 得到0-9之间的随机数
  • 浮点数转化为整数。理论上数字类型都可以转化，只是自动去掉小数点

  • 示例：生成验证码：

    double value = Math.random();int code = (int)(value*1000000);  //生成6位验证码❌。因为可能只有5位数

double value = Math.random();int code = (int)((value+1)*100000);  //生成6位验证码。确保是6位，但第1位总是1。❌

int code = (int)((Math.random()*9+1)*1000);   //生成4位验证码。4个数均随机。✔

自定义方法

（自己添加的方法，如基础的 main 方法：main方法有特殊含义：是程序执行的入口，用于启动程序。如果要执行代码必须要有main方法）

    // public（公共的） static（静态） void（空类型）
        public static void 方法名称(方法参数){
          代码块
        }

    eg：public class Custom {
    public static void main(String[] args){   //Java执行程序的方法
      System.out.println("程序从这开始执行");
    }
}

  • Java命名规范（遵循“驼峰式”）：

    • 方法名称：小驼峰：第一个单词首字母小写，第二个大写 eg：myDocument、password
    • 大驼峰：每个具体单词首字母大写 eg：UserName

方法调用

方法创建成功后：

  • 自定义的方法都在class类当中，顺序在main方法前或后都可
  • 但方法内，不能新建方法
  • 自定义方法可以被调用多次，方法内也可以调用其他方法

为什么需要方法：

  • 解决代码复用

  • 隐藏程序细节，使程序更清晰，从而降低复杂度

  • 代码解耦合，支持多人协同开发

    public class Custom {

  public static void main(String[] args){
    System.out.println("测试");
    newLine();
    newLine();    //调用多次
    System.out.println("结束");
  }

  public static void newLine(){
    System.out.println("123");
    format();    //调用下面的 format 方法（方法内也可以调用其他方法）
  }

  public static void format(){
    System.out.println("------------");
  }

}

结果：
    测试
    123
    ------------
    123
    ------------
    结束

方法参数

  • 参数类型 参数变量名

    • 多参数方法：定义多个参数，用“ ,“ 分隔

  • 实际上，方法参数 和 声明变量 并无区别

  • 形参：编程语言把在方法参数位置声明的变量叫做形参

    public class MessageCode {

  public static void main (String[] args) {
    String text = "亲爱的XXX，你本次的验证码是";
    code(text,1000);
    code(text,100000);    //逗号分隔
  }

  public static void code(tring text,int length){    //多个方法参数
    int code = (int)((Math.random()*9+1)*length);
    System.out.println("亲爱的XXX，你本次的验证码是【"+code+"】");
  }

}

结果：
        亲爱的XXX，你本次的验证码是【1127】
        亲爱的XXX，你本次的验证码是【148083】

  • 方法有了参数声明后，就必须要传入参数，否则编译失败

  • 实参：把传入的参数，可以是变量，叫做实参

    random(100000);    // 直接传值

int len = 100000;    // 传递变量
random(len);
