区别

有返回值方法 & 无返回值方法 的区别

  • 方法返回值类型的声明不同
    • void 表示无返回值，没有 void 表示有返回值
  • 方法返回语句不同
    • 有返回值方法执行时，必须执行 return 返回结果;
      • 且返回结果类型必须和方法声明的返回值一致
      • 且 return 返回语句基本在程序的最后一行

eg：count值是4时返回4位验证码，其他值返回6位验证码：

public class Test52 {

  public static void main(String[] args) {

    int captchaCode = code(4);    // 得到4位随机码
    System.out.println("本次的验证码是:" + captchaCode);    //2053

    captchaCode = code(8);    // 得到6位随机码
    System.out.println("本次的验证码是:" + captchaCode);    //955620
  }

  /**
   * 得到验证码
   */
  public static int code(int count) {
    if (count == 4){
      return (int)((Math.random()*9+1)*1000);
    }else {
    return (int)((Math.random()*9+1)*100000);
    }
  }

}

布尔

  • 关系运算符：结果为true/false（布尔类型）

  • boolean 类型：可作为方法的参数or返回类型，如判断是否及格

    public class Test53 {

  public static void main(String[] args) {
    System.out.println(isPass(80));

    if(isPass(70)){
      System.out.println("及格啦");
    }
  }

  public static boolean isPass(int score){
    return score >= 60;
  }
}

逻辑运算符

  • 与：&&、或：|| 、非： !

  • 作用：组合多个条件

  • 可对 boolean数据 进行 ! 的运用，表示否定 boolean 值。eg：

    boolean isPass = false;

if(!isPass){
  System.out.println("通过考试");

eg：判断是否是个位数

public class Test55 {

  public static void main(String[] args) {
     if(isSingleDigit(12)) {
        System.out.println("是个位数");
     }else{
        System.out.println("这不是个位数");    //传入12，返回值为false。故输出"这不是个位数"
     }
  }

  public static boolean isSingleDigit(int x){
     return x>=0 && x<10;
  }

}
