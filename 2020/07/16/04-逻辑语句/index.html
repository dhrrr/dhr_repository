if 条件语句

  • 计算机的世界除了0和1，就是一堆 if else 的集成

  • 条件语句：执行条件判断的语法

  • if 条件语句：最简单的条件形式

    public class ConditionTest1 {
    public static void main(String[] args) {
    sayX(20);
}
public static void sayX(int x){
    if(x>0){
        System.out.println("x：" +x);
        }
    }
}
  • if 后面的小括号叫做：条件表达式 。若为真（成立），则执行大括号中的语句；若不成立，则什么都不会发生
    • 条件表达式可包含任意比较运算符（or 关系运算符）
    • 比较运算符左右对象类型必须一样，int只能和int，double只能和double
      • x == y // x等于y 。（”==”：判断是否相同 “=” ：赋值运算符）
      • x != y // x不等于y
      • x > y
      • x < y
      • x >= y // x =>/=< y 是错误写法
      • x <= y

if else 选择性语句

  • 选择性执行，意为有两条可选路径中选择一条

    public class Condition{
      public static void main(String[] args){
        number(4);
            number(5);
      }
      public static void number(int x){
        if(x % 2 == 0){
              System.out.println(x+"是偶数");
        }else{
              System.out.println(x+"是奇数");
        }
      }
}    

结果： 4是偶数 5是奇数

条件判断链

  • 条件判断多于两个

  • 根据条件不同，执行不同的分支

  • else if 和 if 没有本质区别，唯一区别是执行顺序，从第一个条件判断开始，若不成功则继续下一个判断，一直到没有条件符合的情况

  • 潜在逻辑：条件链也是选择性执行，条件符合后，只会执行符合的分支代码，不是所有都执行

    public class ConditionTest9 {

  public static void main(String[] args) {
    schedule(7.0);
    schedule(13.3);
    schedule(23.0);
  }

  public static void schedule(double time){
    if(time == 7.0){
      System.out.println("7:00 起床了");
    }else if(time == 13.30){
      System.out.println("13:30 午睡时间");
    }else if(time == 23.0){
      System.out.println("23:00 睡觉了");
    }
  }

}

嵌套语句

  • Java一行一行执行，条件语句基本都是选择性执行

  • 符合一个情况后再去判断条件（嵌套条件）

  • 嵌套执行：if 语句里再执行 if else 语句

  • Java可多层嵌套

    public class ConditionTest11 {

  public static void main(String[] args) {
    student("小A",4,426);
    student("小B",4,300);
    student("小C",4,500);
    student("小D",6,400);
    student("小E",6,505);
  }

  public static void student(String name,int level,int grade){
    if(grade >= 425){
        if(level == 4){
          System.out.println("恭喜你通过英语四级考试");
        }if(level == 6){
          System.out.println("恭喜你通过英语六级考试");
        }
      System.out.println(name+grade);
    }else{
      System.out.println(name+grade);
    }
  }

}

结果：恭喜你通过英语四级考试
      小A426
      小B300
      恭喜你通过英语四级考试
      小C500
      小D400
      恭喜你通过英语六级考试
      小E505

返回语句

  • 用返回语句 return; 目的：中断程序执行

  • 执行到 return;会中断后面的代码

  • return;语句之后不能有其他代码

    public static void main(String[] args) {
    plan(39);
  }

public static void plan(double temperature){
    System.out.println("准备出门去学校");

    if(temperature>38){
       return;    // 体温超过38度，属于发烧，忌运动
    }
    System.out.println("去操场踢球");
  }
}

结果：准备出门去学校

递归

  • 程序设计重要思想

  • 打破一行一行呆板执行

  • 方法调用自身，这个方法称为递归方法

    public class CountDown {

  public static void main(String[] args) {
    count(3);
  }

  public static void count(int number) {
    System.out.println(number);
    number = number - 1;

    if (number == 0) {
      System.out.println("程序执行完毕");
      return;
    }
    count(number);

  }

}

结果： 3
      2
      1
      程序执行完毕    //倒计时功能
