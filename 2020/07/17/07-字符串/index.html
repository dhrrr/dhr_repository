调用字符串对象方法

  • 在Java或其他一些编程语言，对象表示具有相关性的数据和方法的集合
  • 方法用来操作对象、执行计算和修改对象数据等
  • Java中字符串String也是对象，故也包含数据和方法

length()

int size = message.length();

charAt()

  • charAt 方法 返回的数据类型为 char

  • char 是单个字符类型，故 charAt方法 只能存储一个字符

    str = message.charAt(1);

  • 但可结合 length()、for 语句得到每个（多个）字符

    public static void main(String[] args) {
    print("今天天气挺好的");
  }

  public static void print(String message){
    for (int i = 0; i < message.length(); i++) {
      char str =  message.charAt(i);
      System.out.println(str);    //输出：今天天气挺好的
    }
  }

trim()

  • trim() 方法执行后会得到一个新字符串，故要重新声明一个字符串
  • 会把左右两边空格都清除，不管有多少空格

    String newStr = str.trim();

字符串常用方法

indexOf()

  • 找出一段话中是否有某个词，若有，查找第一个匹配到的坐标索引值（int类型）

  • 返回值是 -1 则不匹配，返回值 != -1 则匹配到了。因为匹配到时，一定有下标，不可能为负数

    int index = str.indexOf(“Java”);

    public static void main(String[] args) {

    String str = "Java是一种广泛使用的计算机编程语言，拥有跨平台、面向对象、泛型编程的特性，广泛应用于企业级Web应用开发和移动应用开发。";

    int index = str.indexOf("Java");

    if(index != -1){
      System.out.println("匹配到了Java，索引位置是"+index);
    }else{
      System.out.println("没有匹配到Java");
    }
  }

  • indexOf(“xxx”, 开始索引值 )：设定从什么位置开始查找

  • 若要查找第二次出现Java的位置，则 开始索引值 应为 第一次匹配到的索引值 + 要查找的字符串的长度

    index = str.indexOf(“Java”, index + 4);

substring()

  • 得到某个匹配到的字符之后的内容。有两种调用方式：

    • substring( 开始索引 )：从开始索引开始（包含该字符）一直到结束
    • substring( 开始索引 , 结束索引 )：从开始索引开始（包含该字符）到结束的索引结束（不包含该字符）

      String newStr = str.substring(index+4,index+7);

  • 有时需要计算一下结束索引：

    public class Test78 {

  public static void main(String[] args) {
    String str = "Java是一种广泛使用的计算机编程语言。";
    int index = str.indexOf("Java");

    if (index != -1) {
      System.out.println("匹配到了Java，索引位置是" + index);

      String newStr = str.substring(index+4,index+7);    //截取Java后面的三个字符。开始索引是：index+4，结束索引是：index+4+3
      System.out.println(newStr);    //输出：是一种

    } else {
      System.out.println("没有匹配到了Java");
    }
  }

}

startsWith() / endsWith()

  • 判断是否以 某字符串 开始 或 结束，故返回类型为 boolean
  • 用于 if 语句中

    fileName.endsWith(“.doc”)

public static void main(String[] args) {

    String fileName = "报告.doc";
    String url = "https://www.youkeda.com";

    if(fileName.endsWith(".doc")){    //返回值为true，故输出下一句
      System.out.println("是word文档");
    }

    if(url.startsWith("https")){
      System.out.println("网址是安全的");    //返回值为true
    }

  }

replaceAll()

  • 有两个字符串参数，语法：replaceAll( “要替换的字符” , “新字符” )
  • 执行后得到一个新字符串。原始字符串不会发生变化
  • 也可用作删除内容，把 新字符 换成 空格 即可

String newStr = str.replaceAll(“Java”,”Python”);

String name = fileName.replaceAll(“.doc”,””);

字符串操作

split()

  • 处理类似上图有格式的文本数据时，先将文本数据转变为数组数据，即需要 split(“分割字符”)
    • 该处的分割字符可为字符串或特殊字符，如：换行符 \n 。且换行符在文本文件是看不见的
    • 注：这三个特殊字符 . 、| 、* 作为分隔符时，需加上 \，如 str.split(“\|”)

toUpperCase() / toLowerCase()

  • 统一格式输出。作用：把字符串中字母全变为大/小写，并返回出 新字符串数据
  • 大小写转化方法无参数

    String enName = text.toUpperCase();

equals()

  • 用于比较两个字符串是否相同
  • 并不是用 == 表达式 ❌。但Java的jvm编译做了优化后，有时使用 == 运行结果也会成功

Interger.parseInt() / String.valueOf()

  • 对字符串中的数字进行数学运用

    • 将字符串变为数字：Interger.parseInt()
    • 将数字变为字符串：F1：String.valueOf() 强制转化；F2：数字+空字符串
      • valueOf 方法参数接受 数字、浮点、布尔类型 转化为字符串
      • 空字符串相加数字，自动变成字符串类型

  • 字符串转数字举例：

    String text = “123”;

    int a = Integer.parseInt(text);

  • 数字转字符串举例：

    int a = 100;

    String str1 = “” + a;
    String str2 = String.valueOf(a);
