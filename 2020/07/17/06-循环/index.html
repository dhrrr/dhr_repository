自增 & 自减

  • 自增：
    • 自己加自己，默认+1，故 1 可省略
    • 语法：变量名称++ eg：number++，等同 number=number+1;
  • 自减：
    • 自己减自己，默认-1，1 省略
    • 语法：变量名称– eg：number–，等同 number=number-1;
  • 其他步长：
    • 固定加减数字都可采用自增和自减的方法
    • 如：number = number - 2，可写为 number -= 2

eg：

public class Test61 {

  public static void main(String[] args) {
    int a = 10;

    a++;
    System.out.println(a);    // 11

    a+=2;
    System.out.println(a);    // 13
  }

}

数组

  • 也是一个基础数据类型（还有字符串、数字、布尔）
  • 数组：一组类型相同的数据组合在一起
  • very important 数据结构

数组的创建

int[] 整型数组类型、 String[] 字符串数组类型

  • int[] 整型数组类型、 String[] 字符串数组类型
  • 因为数组就是一堆相同类型的数据的集合，故大小要确定
  • 创建好数组后，还需存储每一个数据，需借助下标，且下标不能超过数组的长度。0 代表第一个数据
    • 存储数据的简写方式：eg：String[] str = new Str[]{“张三”,”李四”, “王五”}; or int[] num = newNum[]{1,2,3};
      • 虽方便，但只有在数据很确定时才可简写。同时完成了数组长度的设定和数组的赋值
      • new String[]{“张三”,”李四”, “王五”}; 表示初始化字符串数组数据

获取数组的某一个数据

  • 同样用 下标

    public class Test61 {

  public static void main(String[] args) {
    int[] numbers = new int[3];    // 声明一个 int 数组的变量，数组大小为6

    numbers[0] = 1;
    numbers[1] = 3;
    numbers[2] = 5;

    int num = numbers[0];
    System.out.println(num);    // 1

    num = numbers[1];
    System.out.println(num);    // 3

  }

}

数组的长度

  • 数组对象都有一个值：length，则改数组的最后一个数据的下标为： length-1

  • 数组越界问题：

    public static void main(String[] args) {
  int[] numbers = new int[8];

  int size = numbers.length;
  System.out.println(size);    // 数组长度为；8

  System.out.println(numbers[size]);    //控制台报 数组越界 错误。因为下标只能到7：number[7]。故number[size] ❌
}

对数组初始化

  • 作用：开辟该长度的存储空间

  • 数组初始化后，并没有存储实际值，故未赋值时， int[ ] 的每一个值都默认为 0 ，String[ ] 的每一个值默认都为 null

    public class Test61 {

  public static void main(String[] args) {
    int[] numbers = new int[8];    //初始化变量
    System.out.println(numbers[0]);    // 0

    String[] strs = new String[2];    //初始化变量
    System.out.println(strs[1]);    // null
  }

}

for 循环

  • 遍历分析数组中的每一项数据。自动执行计数器从0开始递增到指定值为止

  • for 语句中小括号的内容 指定了计数器如何工作

  • 以下两种循环等同：

    写法1：
public class Test61 {

  public static void main(String[] args) {
    String[] tables = new String[3];
    tables[0] = "张三";
    tables[1] = "李四";
    tables[2] = "王五";

    for(int i = 0; i < tables.length; i++){
      String name = tables[i];
      System.out.println(i+"："+name);
    }
  }

}

    • int i=0; 声明了一个新计数器，从0开始工作

    • i< tables.length; 声明计数器的条件，最大不能超过 tables.length

    • i++ 声明计数器的工作机制，这里为 递增，每次 +1

      扩展写法2：
for (String name : tables) {

}

嵌套循环

  • eg：打印 9*9 乘法口诀表

    public class Test67 {

  public static void main(String[] args) {
     for(int i=1; i<=9; i++){    //控制行
        String lines = "";
        for(int j=1; j<=i; j++){    //每一行的最大乘法不会超过该行行数
           lines = lines + j + "*" + i + "=" + j*i +" ";    //拼接字符串，打印出乘法口诀表效果
        }
        System.out.println(lines);
     }
  }

}

结果：

  • 第一个for循环目的：控制行
  • 第二个for循环目的：完成每一行的数据
