文档结构

<!DOCTYPE html>

  • 作用：告知浏览器该页面文件的文档类型
  • 必须在第一行。大小写不敏感。无结束标签

<html lang="en">......</html>

  • 作用：告知浏览器自身是一个HTML文档
  • lang属性：语言属性
    • 标签属性
    • 一个标签可有0个or多个标签属性
    • 标签属性与标签名称、标签属性与标签属性 之间需用空格隔开
    • 可赋予标签更多信息。形式：名称=”值”（key=”value”）

<head>......</head>

  • 一个<head>元素必须包含且只能包含一个<title>元素

<body>......</body>

  • 包含文档所有内容
    • 文本、超链接、图像、表格、列表 etc

元素结构

  • 一个 HTML元素 由 开始标签、内容、结束标签 构成
  • 通常成对出现。也有单独呈现的标签。eg：<input>、<img> 等
  • 统一要求为 小写

嵌套

  • 在HTML中，元素可发生嵌套
  • div元素是p元素的 父元素 ，p元素是div元素的 子元素

文本标签

1、常用文本标签

  • 段落标签：
    <p> </p>
  • 内联标签：
    • <span> </span>
      • 通常表示一个行内短语，使网页更有条理。未加样式前，样式上与普通文本无区别
    • <strong> </strong>
      • 加粗词句，重点标注

2、标题标签

  • 数字越大、级别越低、标题越小
  • 下一级标题是上一级标题的子标题，一级标题后可有多个二级标题
  • 标题之间不可越级

3、图片标签

<img src="/P3-1-HTML-CSS/1.3/1-3-island-2.jpg" alt="" />
  • 内联标签，且无结束标签
  • 属性：
    • src：中是图片的 相对路径或绝对路径
    • alt：定义了描述图片的替换文本，若图片url错误，该图片不在支持的格式列表中，或图像还未被下载，用户将看到此处的替换文本

4、链接标签

<a href="https://www.google.com/">谷歌搜索</a>
  • 是内联标签
  • 属性：
    • href：给出链接指向的网站，值为一个URL或锚点
    • title：给出链接的说明信息
    • target：属性值可为 _self _blank _parent _top
      • _self：表示在当前页面打开 _blank：表示在新页面打开
