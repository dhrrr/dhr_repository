认识前端开发

发表于 2020-08-02 | 分类于 Web前端基础
字数统计: 614 | 阅读时长 ≈ 2
  • 前端开发是创建Web页面或app等前端页面并呈现给用户的过程
  • 通过 HTML，CSS，JavaScript 衍生出的各种技术、框架、解决方案，实现互联网产品的用户界面交互
  • 前端开发由网页制作演变而来
    • 早期会Photoshop和Dreamweaver即可制作网页，为Web1.0时代
    • 2005年后，互联网进入Web2.0时代
  • 前端技术 包括：
    • 前端页面设计师、浏览器兼容
    • CSS、HTML、Adobe AIR、Google Gears
    • 概念性较强的交互式设计、艺术性较强的视觉设计 etc
  • 前端三把斧

HTML

  • 全称：HyperText Maekup Language 超文本标记语言
  • 蒂姆 · 伯纳 ·斯里 提出并创建
  • 描述一个网页结构。是一种标记语言，非编程语言
  • 一个HTML元素就是一个HTML文件的一个基本组成单元，是构建网站的基石
  • 浏览器使用 HTML标签（<html>） 和 脚本（JS）诠释网页内容，但不会将标签显示在网页上
  • 发展历史：

CSS

  • 全称：Cascading Style Sheets 层叠样式表
  • 又称：串样式列表、级联样式表、串接样式表、阶层式样式表
  • 用于为结构化文档（eg HTML文档 or XML应用）添加样式（字体、间距、颜色 etc）
  • 计算机语言。职能：确定布局与元素表现。简化HTML文件，又使CSS文件更易维护
  • 不可单独使用，必须与HTML或XML协同工作。起装饰作用
  • CSS使页面内容与样式得以分离：
    • HTML文件只包含结构和内容信息
    • CSS只包含样式信息

JavaScript

  • 是一种高级、解释性的编程语言。简称 JS
  • 使网页可交互
    • 复杂动画、可点击按钮、通俗菜单、表单验证、页面特效、操作页面元素
  • 支持面向对象编程、命令式编程、函数式编程
  • 提供语法来操控文本、数组、日期、正则表达式 etc
  • 已由 ECMA（欧洲计算机制造协会）通过 ECMAScript 实现语言的标准化
  • JS 与 Java 是两门不同的编程语言：
    • JS 的语言设计受 Self（一种基于原型的编程语言）和 Scheme（一门函数式编程语言）影响。是运行在浏览器中的一种 动态、弱类型 脚本语言
    • Java 是一种可撰写跨平台应用软件的面向对象的 静态、强类型 程序设计语言
