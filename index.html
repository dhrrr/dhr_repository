数学函数（生成随机数）：

double value = Math.random();    //0.40068331197074114   0.21177156791789276

  • Math.random()方法用于返回一个随机数

  • 随机数范围为：0.0=<X<1.0

    double value = Math.random();
int newValue = (int)(value * 10);    //把随机的结果*10 得到0-9之间的随机数
  • 浮点数转化为整数。理论上数字类型都可以转化，只是自动去掉小数点

  • 示例：生成验证码：

    double value = Math.random();int code = (int)(value*1000000);  //生成6位验证码❌。因为可能只有5位数

double value = Math.random();int code = (int)((value+1)*100000);  //生成6位验证码。确保是6位，但第1位总是1。❌

int code = (int)((Math.random()*9+1)*1000);   //生成4位验证码。4个数均随机。✔

方法调用

方法创建成功后：

  • 自定义的方法都在class类当中，顺序在main方法前或后都可
  • 但方法内，不能新建方法
  • 自定义方法可以被调用多次，方法内也可以调用其他方法

为什么需要方法：

  • 解决代码复用

  • 隐藏程序细节，使程序更清晰，从而降低复杂度

  • 代码解耦合，支持多人协同开发

}

Java如何运行 & 注释

发表于 2020-07-22
  • Java：面向对象的程序设计语言。目前属于Oracle公司
  • 高级语言：Java、Python、C、C++等 低级语言又叫机器语言or汇编语言 计算机只能运行低级语言，故高级语言编写的程序必须先被翻译成低级语言才能运行
  • 高级语言（称为源代码）：优点：可移植性。不修改的情况下可以运行多种计算机平台
  • Java代码不能直接被执行
  • 目前编程语言的翻译主要有两种方式：解释（interpreting） & 编译（compiling）
    • 解释器：完成解释任务的程序（按行读取，先翻译后执行命令）
    • 编译器：完成编译任务的程序（在执行之前一次性对程序进行完整编译，故程序的运行是在编译过程之后，编译器生成的程序称为可执行程序）
  • Java既可以被编译，也可以被解释
  • Java编译过程中，不生成机器语言，而是生成字节码（字节码具备高级语言的可移植性）
  • 除了生成字节码之外，Java还创建了JVM（Java虚拟机），通过JVM支持字节码的解释运行
  • Java源代码是一个后缀为java的文本文件。文件名必须是英文字母且一般首字母大写，尽量不用数字。eg：HelloWorld.java
  • Java代码的注释：
    • 行注释：//xxx
    • 块注释：
      • 一般在Java类的声明上、方法上使用块注释

对用户行为日志分析

发表于 2020-07-09

行为日志不仅包含用户的行为，也包含服务器响应是否正常，这样可以快速发现问题并处理。
所以，我觉得行为日志非常重要，观察和分析日志行为是一个必备技能！

有三个基本操作：

  • grep -c ‘11.176.159.95’ cookie_log.log
    //统计关键字出现的次数
  • grep ‘11.176.159.95’ cookie_log.log
    //统计关键字所在行
  • cat cookie_log.log | awk ‘{print $8}’ | sort | uniq -c | sort -k1r | head -10
    //统计出现最多的内容。本例取第8列内容，取前10行
